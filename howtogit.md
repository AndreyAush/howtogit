# подсказки по git

Cоздание репозитория
```sh
git init 
```
Добавление расширение файла
```sh
git add
```
Фиксация событий c описанием
```sh
git commit -m ""
```
Подробный журнал всех изменений
```sh
git log
```
Сокращенный журнал всех изменений
```sh
git log --onelinegit stastu
```
Открыть определенный commit
```sh
git checkout
```
